## LegalFlow3について

LegalFlow3は、法律事務所向けに開発された包括的なケース管理システムです。ケースの管理、当事者情報の整理、ドキュメントの管理、時間追跡、通知機能などを統合し、法律事務所の業務効率を大幅に向上させます。

### 主な機能

- **ケース管理**: 事件の詳細情報、ステータス、優先度の管理
- **当事者管理**: クライアントと相手方の情報管理
- **ドキュメント管理**: 書類のアップロード、分類、検索
- **タイムシート管理**: 作業時間の記録と請求管理
- **通知システム**: 重要な更新や期限の通知
- **カレンダー連携**: Google Calendar、Outlook Calendarとの連携
- **レポート機能**: 業務統計とレポートの生成

## システム要件

### 推奨ブラウザ

- Google Chrome (最新版)
- Mozilla Firefox (最新版)
- Microsoft Edge (最新版)
- Safari (最新版)

### 必要な機能

- JavaScript が有効になっていること
- クッキーが有効になっていること
- ポップアップブロックが無効になっていること

## ログインとアカウント設定

### 初回ログイン

1. 管理者から提供されたログイン情報を使用してログインします
2. 初回ログイン時は、パスワードの変更を求められる場合があります
3. 必要に応じて、プロフィール情報を更新してください

### パスワードの変更

1. 右上のユーザーアイコンをクリック
2. 「設定」を選択
3. 「パスワード変更」をクリック
4. 現在のパスワードと新しいパスワードを入力
5. 「保存」をクリック

### プロフィールの更新

1. 右上のユーザーアイコンをクリック
2. 「設定」を選択
3. 必要な情報を更新
4. 「保存」をクリック

## ダッシュボード

ダッシュボードは、システムのメインページで、重要な情報を一目で確認できます。

### ダッシュボードの構成

- **統計カード**: 総ケース数、アクティブケース数、クライアント数などの統計情報
- **チャート**: ケースのステータス分布、カテゴリ分布、収益推移などの視覚的表示
- **最近のアクティビティ**: システム内で発生した最近のアクティビティ
- **今後の期限**: 近づいている期限とタスク

### ダッシュボードの更新

- ページを再読み込みすると最新の情報が表示されます
- 自動更新機能により、定期的にデータが更新されます

## ケース管理

### ケースの作成

1. 左サイドバーの「ケース管理」をクリック
2. 「新しいケース」ボタンをクリック
3. 必要な情報を入力：
   - ケースタイトル（必須）
   - 説明
   - カテゴリ（民事、刑事、家事、企業法務など）
   - ステータス（アクティブ、完了、保留など）
   - 優先度（低、中、高、緊急）
   - クライアント情報
   - 担当弁護士
   - ケース番号
   - 申立日
   - 予想解決日
4. 「保存」をクリック

### ケースの編集

1. ケース一覧から編集したいケースをクリック
2. 「編集」ボタンをクリック
3. 必要な情報を更新
4. 「保存」をクリック

### ケースの検索とフィルタリング

1. ケース一覧ページの検索ボックスにキーワードを入力
2. フィルターオプションを使用：
   - ステータスでフィルター
   - カテゴリでフィルター
   - 優先度でフィルター
   - 日付範囲でフィルター

### ケースの削除

1. ケース詳細ページで「削除」ボタンをクリック
2. 確認ダイアログで「削除」をクリック

**注意**: ケースを削除すると、関連するすべてのデータ（当事者、ドキュメント、タイムシートなど）も削除されます。

## 当事者管理

### 当事者の追加

1. 左サイドバーの「当事者管理」をクリック
2. 「新しい当事者」ボタンをクリック
3. 当事者のタイプを選択：
   - **個人**: 個人のクライアントや相手方
   - **法人**: 企業や団体
4. 必要な情報を入力：
   - 名前（必須）
   - メールアドレス
   - 電話番号
   - 住所
   - 関連ケース
5. 「保存」をクリック

### 当事者の編集

1. 当事者一覧から編集したい当事者をクリック
2. 「編集」ボタンをクリック
3. 必要な情報を更新
4. 「保存」をクリック

### 当事者の検索

1. 当事者一覧ページの検索ボックスにキーワードを入力
2. フィルターオプションを使用：
   - タイプでフィルター（個人/法人）
   - 関連ケースでフィルター

## ドキュメント管理

### ドキュメントのアップロード

1. 左サイドバーの「ドキュメント」をクリック
2. 「アップロード」ボタンをクリック
3. ファイルを選択（複数選択可能）
4. ドキュメント情報を入力：
   - タイトル（必須）
   - タイプ（契約書、メモ、訴状、証拠など）
   - 関連ケース
   - 説明
5. 「アップロード」をクリック

### ドキュメントの表示

1. ドキュメント一覧から表示したいドキュメントをクリック
2. ドキュメントの詳細情報が表示されます
3. 「ダウンロード」ボタンでファイルをダウンロードできます

### ドキュメントの検索

1. ドキュメント一覧ページの検索ボックスにキーワードを入力
2. フィルターオプションを使用：
   - タイプでフィルター
   - 関連ケースでフィルター
   - アップロード日でフィルター

### ドキュメントの削除

1. ドキュメント詳細ページで「削除」ボタンをクリック
2. 確認ダイアログで「削除」をクリック

## タイムシート管理

### タイムシートエントリの作成

1. 左サイドバーの「タイムシート」をクリック
2. 「新しいエントリ」ボタンをクリック
3. 必要な情報を入力：
   - 関連ケース（必須）
   - 作業内容（必須）
   - 開始時間
   - 終了時間
   - 請求可能時間
   - 時給
4. 「保存」をクリック

### タイマー機能

1. タイムシートページの「タイマー開始」ボタンをクリック
2. 作業内容を入力
3. 関連ケースを選択
4. タイマーが開始されます
5. 作業完了時に「停止」ボタンをクリック
6. 自動的にタイムシートエントリが作成されます

### タイムシートの編集

1. タイムシート一覧から編集したいエントリをクリック
2. 「編集」ボタンをクリック
3. 必要な情報を更新
4. 「保存」をクリック

### タイムシートの削除

1. タイムシート詳細ページで「削除」ボタンをクリック
2. 確認ダイアログで「削除」をクリック

## 通知設定

### 通知の確認

1. 左サイドバーの「通知」をクリック
2. 通知一覧で未読の通知を確認
3. 通知をクリックして詳細を表示
4. 「既読にする」ボタンで既読にできます

### 通知設定の変更

1. 通知ページの「設定」タブをクリック
2. 通知チャンネルを選択：
   - メール通知
   - SMS通知
   - プッシュ通知
   - LINE通知
   - アプリ内通知
3. 通知の種類を選択：
   - ケース更新
   - 期限リマインダー
   - ドキュメントアップロード
   - タイムシートリマインダー
   - システムアラート
4. 通知頻度を設定
5. 静音時間を設定（必要に応じて）
6. 「保存」をクリック

## カレンダー連携

### Google Calendar連携

1. 設定ページの「カレンダー連携」セクションを開く
2. 「Google Calendar連携」をクリック
3. Googleアカウントでログイン
4. 権限を許可
5. 連携が完了します

### Outlook Calendar連携

1. 設定ページの「カレンダー連携」セクションを開く
2. 「Outlook Calendar連携」をクリック
3. Microsoftアカウントでログイン
4. 権限を許可
5. 連携が完了します

### カレンダーイベントの作成

連携後、以下のイベントが自動的にカレンダーに追加されます：

- ケース関連のイベント
- 期限リマインダー
- 法廷での審理
- クライアント面談

## トラブルシューティング

### ログインできない

1. メールアドレスとパスワードが正しいか確認
2. 大文字小文字を確認
3. ブラウザのキャッシュとクッキーをクリア
4. 別のブラウザで試す
5. 管理者に連絡

### ページが表示されない

1. インターネット接続を確認
2. ブラウザを最新版に更新
3. JavaScriptが有効になっているか確認
4. ポップアップブロックが無効になっているか確認

### ファイルがアップロードできない

1. ファイルサイズが制限内か確認（10MB以下）
2. ファイル形式がサポートされているか確認
3. インターネット接続を確認
4. ブラウザを最新版に更新

### 通知が届かない

1. 通知設定を確認
2. メールアドレスが正しいか確認
3. スパムフォルダを確認
4. ブラウザの通知許可を確認

## よくある質問

### Q: 複数のケースを同時に管理できますか？

A: はい、LegalFlow3では無制限にケースを管理できます。検索とフィルタリング機能により、必要なケースを素早く見つけることができます。

### Q: ドキュメントのバージョン管理はできますか？

A: はい、同じファイル名でアップロードすると、新しいバージョンとして保存されます。過去のバージョンも履歴として確認できます。

### Q: タイムシートのデータはエクスポートできますか？

A: はい、タイムシートデータはCSV形式でエクスポートできます。レポート機能からエクスポートできます。

### Q: 複数のユーザーが同時にシステムを使用できますか？

A: はい、複数のユーザーが同時にシステムを使用できます。リアルタイムでデータが同期されます。

### Q: データのバックアップは自動で行われますか？

A: はい、データは定期的に自動バックアップされます。また、手動でバックアップを作成することもできます。

### Q: モバイルデバイスからも使用できますか？

A: はい、レスポンシブデザインにより、スマートフォンやタブレットからも快適に使用できます。

### Q: カスタマイズは可能ですか？

A: 基本的な設定（通知設定、表示設定など）はカスタマイズ可能です。詳細なカスタマイズについては管理者にご相談ください。
